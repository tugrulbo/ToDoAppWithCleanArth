### Simple ToDo App with Clean Arth.

- Simple ToDo app.
- Getting datas from room, inserting todos, searching todos, updating todos

# Contents
- MVVM
- Room
- Recyclerview with diffUtil
- Simple recyclerview anim.
- Navigation
